## @loadable/component

+ [文档](https://loadable-components.com/docs/getting-started/)
+ 用于拆分 react 代码
+ 对 `React.lazy` 的一种补充
  + Full dynamic import
  + SSR
  + with/without Suspense
  + Library splitting
+ `react-loadable` 不再维护, 使用 `React.lazy` 或 `@loadable/component`
