# npm 

## 创建 
+ `npm init -y` `npm init --yes`
+ `package.json`
```json
{
  "name": "test",
   "version": "1.0.0",
   "description": "", // 描述
   "main": "index.js", // 入口文件
   "scripts": { // 支持的脚本
      "test": "echo \"Error: no test specified\" && exit 1"
   },
   "keywords": [],
   "author": "", // 作者
   "license": "ISC" // 许可文件
}
```
